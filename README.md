## Build steps

1. Setup environment (If applicable)
```
nix-shell -p git
```

2. Clone the repository
```
git clone https://gitlab.com/homelab3330409/nix-config.git && cd nix-config
```

3. Switch branch
```
git checkout dendritic
```
