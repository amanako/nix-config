*This branch is made as my attempt to employ dendritic pattern via [den][den] and is still a **WORK IN PROGRESS**  
Current caveat is regarding noctalia binds and fcitx5*

## Build steps

1. Setup environment (If applicable)
```
nix-shell -p git
```

2. Clone the repository
```
git clone https://codeberg.org/abyssal-twilight/nix-config.git && cd nix-config
```

3. Switch branch
```
git checkout den
```
